Automatically fixes the capitalization in post titles. 

== Description ==

This plugin automatically fixes the capitalization in post titles. It checks for small words including 'of','a','the','and','an','or','nor','but','is','if','then','else'
,'when', 'at','from','by','on','off','for','in','out','over','to','into','with' and changes them to lowercase as is normal in the English language. It will update the 
capitilization of all current and new posts when activated. Any new posts that are created while this plugin is activated will have their post title capitalization 
automatically fixed upon submission of the new post.

== Installation ==

1. Upload `fixcaps` folder to the `/wp-content/plugins/` directory.
2. Activate the "Fix Post Title Capitalization" plugin through the 'Plugins' menu in WordPress.
3. Your post title capitalization will automatically be corrected upon activation.
